
Сделать папку репозиторием — git init

«Разгитить» папку, если что-то пошло не так, — rm -rf .git

Проверить состояние репозитория — git status


Подготовить файлы к сохранению — git add имя файла

    git add file.txt

    git add --all # подготовили к сохранению все файлы в репозитории
    git add . # добавить всю текущую папку


Выполнить коммит — git commit 

    git commit -m 'Мой коммит!'


Просмотреть историю коммитов — git log
Получить сокращённый лог — git log --oneline


# команде git log можно указать несколько веток, и тогда она выведет их все
# с флагом --graph  Git нарисует ветки с помощью «палочек» и «звёздочек»

$ git log --graph --oneline main add-docs
